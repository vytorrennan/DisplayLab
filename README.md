## Como rodar o projeto
### Crie e inicie o ambiente virtual
windows:

  `python -m venv .venv`

  `./.venv/Scripts/activate`

linux: 

  `python3 -m venv .venv`

  `source .venv/bin/activate`

### Instale Django
`pip install django`

### Execute o servidor
windows: `python manage.py runserver`

linux: `python3 manage.py runserver`
